如果你的浏览器支持@supports，这行字会是绿色

@supports (display: block) {
    /* 如果支持display: block,则下面指定的CSS会生效 */
    .supports {
    	color: green;
    }
}

检测是否支持指定的CSS属性或者其带前缀版本

@supports ( (-ms-transform: rotate(7deg)) or (-webkit-transform: rotate(7deg)) 
or (transform: rotate(7deg)) or (-o-transform: rotate(7deg)) ) {
    … /* 如果支持不带前缀以及带前缀的transform属性,则下面指定的CSS会生效 */
    .supports-or {
		-webkit-transform: rotate(7deg);
		-ms-transform: rotate(7deg);
		-o-transform: rotate(7deg);
		transform: rotate(7deg);
	}
}

检测是否不支持指定的CSS属性

@supports ( not ((position: fixed)){
    … /* 如果不支持position: fixed */
    position: absolute;
}
